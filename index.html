Desarrollos IA, Add-ins y Plataformas

Aplicaciones propias que conectan modelos BIM, presupuestos, auditorías y control de proyectos. Son desarrollos web y add-ins para Revit construidos con automatización, datos estructurados y flujos asistidos por IA.

Web + Add-in Revit

Quantiva / ItemiCostos

Plataforma para itemizado jerárquico, presupuesto tipo S10, recursos, APU, metrados y fórmulas polinómicas. El add-in se integra con Revit mediante WebView2 para importar costos, verificar codificaciones, crear tablas y exportar metrados BIM al backend.

React TypeScript Node MySQL Revit API

Add-in Revit 2024/2025

JeiTools

Suite de herramientas de apoyo para modelado, CAD, tablas y estándares BIM. Centraliza rutinas frecuentes en una interfaz WPF por pestañas y cuenta con instalador MSI/Burn para despliegue profesional en equipos de producción.

C# WPF .NET WiX BIM Standards

Auditoría BIM / QA-QC

JeiAudit + Auditoría LOIN

Add-ins para revisar existencia de parámetros, nomenclatura de familias, subproyectos, coordenadas, categorías modeladas y reglas LOIN. Generan salidas Excel, CSV y reportes para trazabilidad técnica y tableros de control.

Revit API Model Checker Excel CSV QA/QC

Automatización BIM

Model Auditor y Jobs BIM

Motor de validación y sincronización para codificaciones, metrados, estados de importación y procesos asíncronos. Permite tratar el modelo como fuente de datos y enviar lotes de información a servicios web preparados para flujos BIM en la nube.

APS Workers SSE API REST BIM Data

Gestión de proyectos

MIDP V2

Aplicación para el Modelo Integrado de Desarrollo de Proyectos: configuración, grupos de entregables, control de avance, flujos de revisión, accesos y visualización tipo BI. Está integrada al shell de Dechini Suite y preparada para despliegue en Cloud Run.

JavaScript Node Cloud Run Auth BI

ERP / Presupuestador

ERP Dechini Web

Suite web para propuestas, presupuestos, control comercial, curvas acumuladas, costos reales, valorizaciones y módulos financieros como Ábaco, Pulso y Resumen Comercial. Une frontend React/Vite con backend .NET y persistencia MySQL.

React Vite .NET MySQL GCP
